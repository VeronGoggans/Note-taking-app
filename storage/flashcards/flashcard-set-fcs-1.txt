# flashcard 1
Broken acces control
Toegangscontrole zorgt ervoor dat gebruikers niet buiten hun bedoelde rechten kunnen handelen. Als dit niet het geval is, kunnen gebruikers misbruik maken om ongeautoriseerde functies en gegevens te benaderen.

# flashcard 2
Sensitive data exposure
Het gebruiken van zwakke cryptografie kan leiden tot blootstelling van gevoelige gegevens.
Dit komt door gebruik van zwakke algoritmes en onvoldoende bescherming tijdens verzending.
wrong

# flashcard 3
Injection
injectie fouten zoals SQL injecties, treden op als onbetrouwbare gegevens naar een interpreter worden gestuurd als onderdeel van een opdracht of query. Gebruikers kunnen de interpreter misleiden om onbedoelde opdrachten uit te voeren. 
correct

# flashcard 8
XML external entities
Is een kwetsbaarheid die optreed wanneer een slecht geconfigureerde xml parser externe entiteiten verwerkt die gedefinieerd zijn in een XML bestand. Hierdoor kunnen aanvallers gevoelige data lezen en zelfs RCE (remote control execution) krijgen. 
correct

# flashcard 9
Insecure deserialisatie
Een kwetsbaarheid die optreed wanneer onbetrouwbare of niet-vertrouwde gegevens worden gedeserialiseerd zonder beveiligingscontroles door de applicatie. Waardoor een aanvaller malafide gegevens kan injecteren die leiden tot vershillende aanvalen, zoals RCE.
correct
