Current bugs

1

Name: 
Unicode encoding error

Explanation:
This error occurs, when parts of a note are encoded with a different encoding 
method then the ASCII method

Solution:
Encode the notes with UTF-8 instead of ASCII.
This will ensure a wide range of coverage for different fonts. 

2

Name:
Undeleted notes

Explenation: 
When deleting a folder and or subfolder the corrisponding json gets removed from the notes.json file,
but the actual notes don't get removed from the device.

Solution:
Find out a way to retrieve the folder json to the NoteManager and recursively remove all found notes inside that folder including it's subfolders.
